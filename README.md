# 🧊 WR Gelo — Sistema de Gestão

Sistema de gestão completo para a **WR Gelo Distribuidora**, desenvolvido como uma Single Page Application (SPA) com interface mobile-first.

## 📋 Funcionalidades

- **Dashboard** — Visão geral com KPIs, gráficos de evolução de vendas, mix de clientes
- **Pedidos** — Registro e histórico de vendas com filtros por mês
- **Estoque** — Controle de sacos (3kg, 5kg, 10kg) com movimentações
- **Caixa** — Faturamento, investimento inicial, despesas e breakeven
- **Simulador** — Projeção de cenários com cálculo de ROI e breakeven

## 🚀 Tecnologias

- HTML5 + CSS3 + JavaScript vanilla
- Chart.js para gráficos
- LocalStorage para persistência de dados
- Design responsivo mobile-first

## 🌐 Deploy

O projeto está hospedado no **Vercel**. O deploy é feito automaticamente a cada push na branch `main`.
